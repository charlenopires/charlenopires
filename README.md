## :brain: Who Am I?

> *"I don't write code. I architect cognition."*

I'm a **creative polyglot developer** from **Amarante, Piaui, Brazil** who believes the most interesting problems are the ones nobody has solved yet. My work lives at the intersection of **artificial intelligence**, **systems programming**, and **cognitive architectures** -- building tools that don't just execute instructions, but *reason* about them.

- :jigsaw: **Complexity is my playground** -- I gravitate toward the hardest problems: GraphRAG reasoning, cognitive data structures, multi-agent orchestration
- :globe_with_meridians: **Polyglot by philosophy** -- I choose the right language for each problem: Rust for performance, Elixir for concurrency, Python for ML, TypeScript for reach, Erlang for resilience, Dart for mobile
- :robot: **AI is not a tool, it's a collaborator** -- From LLM-powered development assistants to NARS-based reasoning engines, I build systems where human and machine intelligence amplify each other
- :seedling: **Open source is oxygen** -- 25+ public repositories and counting. Every idea deserves to be shared, challenged, and improved
